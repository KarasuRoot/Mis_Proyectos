# 🛡️ CyberAnalyzer v1.0 - KarasuRØØT

**CyberAnalyzer** es una herramienta integrada de reconocimiento (OSINT), análisis de phishing y verificación de identidades digitales diseñada para analistas de SOC.

## 🚀 Módulos Principales
*   **Infraestructura (OSINT):** Resolución de IPs, escaneo de puertos comunes y detección de tecnologías/SSL.
*   **Phishing Radar:** Análisis heurístico de URLs para detectar suplantación de identidad (Typosquatting).
*   **Identity Investigator:** Extracción y cruce de perfiles en redes sociales (LinkedIn, X, Instagram, Facebook) con detección de patrones de bots.

## 🛠️ Tecnologías Utilizadas
*   **Python 3.14**
*   **Bibliotecas:** `dnspython`, `ipwhois`, `tkinter`, `difflib`.
*   **Compilación:** PyInstaller con optimización de binarios.

## 📋 Uso
1. Descarga el ejecutable desde la sección de **Releases**.
2. Ejecuta `CyberAnalyzer.exe`.
3. Selecciona el módulo deseado y elige entre entrada manual o carga masiva vía `.txt`.

> **Nota:** Esta herramienta fue desarrollada con fines educativos y de auditoría de seguridad. No requiere instalación de Python para su ejecución.
